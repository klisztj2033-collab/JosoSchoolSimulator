# 常総学院シミュレーター

常総学院の入学から最初の3か月（4月〜6月）を体験する、週単位進行の1人用学園シミュレーションWebゲーム。
ビルド不要・サーバー不要。**`index.html` をブラウザで開くだけで動く。**

## 実行方法

### 一番簡単な方法
`index.html` をダブルクリックしてブラウザ（Chrome / Edge / Firefox）で開く。

### ローカルサーバーで開く場合（推奨・セーブが確実に動く）
```
cd C:\JosoSchoolSimulator
python -m http.server 8000
```
→ ブラウザで http://localhost:8000 を開く。

## 遊び方（ときめきメモリアル4スタイル）

1. **タイトル画面** → 「はじめから」
2. **キャラ作成** — 名前・クラス(1〜4組で担任が変わる)・タイプ(初期ステータスが変わる)を選ぶ
3. **コマンド選択画面**で毎週1つコマンドを選ぶ（勉強 / 雑談 / 部活 / 校内探索 / おでかけ / 休養 / イベントを起こす）。
   コマンドにマウスを乗せると下の青いバーに説明が出る
4. イベントは**ADV形式**（全画面背景＋セリフウィンドウ）。クリックでテキスト送り、選択肢で結果が変わる
5. 12週（6月4週）まで進めると **3か月評価**（ランク・称号・親友・記録）が表示される

### 体調システム（ときメモ式）
- 行動するとストレスがたまり、**休養**で回復する
- 体調は「絶好調😄（能力上昇1.25倍）／ふつう🙂／不調😵（0.6倍）」の3段階
- ストレスが限界（100以上）になると**風邪で1週間寝込む**

### その他
- 進行は毎週末に自動セーブ（ブラウザのlocalStorage）。「つづきから」で再開できる。
- 「おでかけ」した週は駅系イベント（荒川沖・土浦）が発生しやすい。
- 「イベントを起こす」と職員室に呼ばれることがある。

## 設計方針

- **1人用・静的Webアプリ**を最小構成で完成させる（HTML/CSS/JS、フレームワークなし）
- **ときめきメモリアル4のシステムを参考** — コマンド選択画面（learn/game01.jpg）、
  ADVイベント画面（learn/game02.jpg）、ストレス・体調システム
- **背景は実画像** — `picture/` の写真を場所別に使用。足りない場面（駅ホーム・電車内）は自作SVG
  （`picture/gen_station.svg` / `picture/gen_train.svg`）。`自宅.jpg` はHEIC形式だったため
  `自宅_conv.jpg` に変換して使用
- **データとエンジンを完全分離** — `js/data.js` にNPC・イベント・行動をすべて集約。
  7月以降への拡張は `WEEK_LABELS` / `WEEKLY_EVENTS` / `RANDOM_EVENTS` に追記するだけ
- **NPCはステータスを持たない**。人物情報（名前・特徴・口癖）と、プレイヤー側が持つ「仲良し度」のみ
- **プレイヤーだけが成長**する（9種ステータス + ストレス + NPC別仲良し度 + フラグ）
- 将来のオンライン化を見据え、ゲーム状態は単一のJSONオブジェクト `S` に集約（そのままサーバー同期に流用可能）

## ファイル構成

```
index.html        画面の土台
css/style.css     スタイル（ときメモ風パステルテーマ + ADV画面）
js/data.js        ゲームデータ（NPC30人 / 固定イベント12週 / ランダムイベント15種 / コマンド7種 / 背景マップ）
js/game.js        エンジン（状態管理・週進行・ADVエンジン・体調システム・セーブ）
picture/          背景画像（実写 + 自作SVG）
learn/            UI参考画像（ときメモ4）
```

## データ構造

### ゲーム状態（セーブ対象）
```js
{
  name, cls, type,        // プレイヤー情報
  week: 0..11,            // 現在週
  stats: { gaku, undo, komyu, ninki, mental, nori, shinrai, renai, omoide },
  stress: 0..120,         // ときメモ式ストレス（体調を決める）
  rel:   { npcId: 0..100 },  // NPC別仲良し度
  flags: { pcclub, group_anime, ... },
  seen:  ["chinone_letter", ...],  // 発生済み1回限りイベント
  log:   ["4月 第1週: 入学式・クラス発表", ...],
}
```

### イベント
```js
{
  title, place,
  text: 文字列 or (state)=>文字列,   // 状態依存テキスト（テスト順位など）
  cond: (state)=>bool,               // ランダムイベントの発生条件
  once: true,                        // 1回限り
  priority: true,                    // 優先発生（必須イベント用）
  choices: [{ label, text, fx: {stat: ±n}, rel: {npcId: ±n}, flag }],
}
```

### 週の流れ
行動選択 → 行動結果 → 固定イベント(選択肢) → ランダムイベント抽選(約6割) → 週末まとめ → 自動セーブ → 次週

## 収録イベント（抜粋）

- 4月: 入学式 / 最初の休み時間 / 部活勧誘(パソコンサークル) / 荒川沖駅・石川西山口論
- 5月: グループ形成 / GW明け / **中間テスト・伊藤けいすけ伝説事件** / 結果発表(順位は学力で変動)
- 6月: 梅雨 / 体育(石川の謎マラソン適性) / パソコンサークル / 3か月評価
- ランダム: **ちのね×大森の手紙** / **荒川沖駅ホーム・小沼あいの意外な一面** / 石川vs西山 /
  久野の大声 / ジブリ語り / ガンダム談義 / 土浦寄り道 / 伊藤の奇行 / 赤上×小沼 ほか
